# MiaoWallet Demo — English Voiceover Script (~3 min)

---

## [Scene 1 — Intro, 20s]
*(打开 cryptomiao.wal.app)*

"Hi, I'm CryptoMiao. Welcome to MiaoWallet — a secure middleware that lets AI agents operate on the Sui network, without ever touching your private keys.

The core idea is simple: give AI the power to act on-chain, but keep the keys strictly offline."

## [Scene 2 — WebGUI, 20s]
*(打开 WebGUI)*

"Here's the control panel. As the owner, you manage wallets and set session limits based on time or signature count. Keys are safely stored in your macOS Keychain, meaning the AI agent never actually sees them. And if needed, you can revoke access instantly."

## [Scene 3 — API Mode, 60s]
*(Telegram 对话)*

"Let's start with API mode. I'll ask my AI agent to perform a few tasks.

First, check the balance... Done.

Next, let's transfer 0.01 SUI to 'bvlgari.sui'. Watch closely: the agent calls MiaoWallet, the Keychain securely signs the request, and it's submitted to the mainnet. Done.

Now, let's swap 0.5 SUI for WAL using Cetus.

Finally, we store this transaction record on Walrus. This creates an immutable attestation, making every AI action fully auditable."

## [Scene 4 — Browser Mode, 45s]
*(浏览器 RPS DApp)*

"Now let's look at Browser mode. Here, the AI controls a real browser using OpenClaw's Browser Relay.

It navigates to our Rock-Paper-Scissors DApp, picks a move, and places a bet. Notice how our Chrome extension — Sui Agent Wallet — intercepts the signing request and securely routes it through MiaoWallet.

You get the exact same security on any DApp, with zero integration required."

## [Scene 5 — Security Features, 25s]
*(WebGUI session + 白名单面板)*

"Let's highlight the security layer. We use a strict whitelist system: only approved DApp origins and smart contracts can even request a signature. Anything else is blocked at the door.

Combined with auto-expiring sessions and a one-click kill switch, the agent's access is completely in your control."
